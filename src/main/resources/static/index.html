抽奖机制

当今社会游戏产业比较发达，多数游戏都有着抽奖机制，玩家可以通过抽奖来获得自己想要的游戏物品，并能从中获得乐趣。

无法满足

由于获奖几率的限制，大多数玩家并不能够获得自己想要的物品。让玩家体验获得高品质物品的乐趣，为此而生。

体验“神抽”

鉴于目前多数玩家在游戏中开包一直获得不到自己想要的卡牌或者高品质的卡牌的前提下，让玩家体验“神抽”的快感。

轻量全面

致力于让玩家体验到平时在真实游戏中体验不到的快感。以及提供详细的卡牌查询服务，让玩家能对游戏卡牌有更深的理解。

我们一起发掘有趣的故事

在知识爆炸的年代，我们不愿成为知识的过客，拥抱游戏文化，发挥玩家们的力量，参与到故事客栈获得自我升华。
